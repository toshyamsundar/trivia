# trivia

Trivia Game

```
The trivia starts with the options to choose the category and difficulty level
Make the choices and get started
There will be 10 questions in total
You will be given 15 seconds for each question
If the correct answer was chosen, it will be highlighted in GREEN
If a wrong answer was chosen, then it will be highlighted in RED
If nothing is chosen within 15 seconds, the correct answer will be highlighted in GREEN
After all 10 questions, a summary page will be displayed with the results
Try the HARD level if you can :)
```
